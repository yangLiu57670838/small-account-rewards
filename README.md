# Software requirements
* postgresql
* Node.js 8
* npm 5


# Installation instructions
* brew install postgresql
* npm install nodemon mocha -g

* Please run the sql commands in ./database/init.sql before you run this program or tests

# Run
npm start

# Run test files
npm test
